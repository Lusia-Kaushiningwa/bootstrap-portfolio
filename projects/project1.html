My Portfolio

Personal Developer Portfolio

🌟 Project Overview

This project is my personal portfolio website designed to showcase my skills, projects, and growth as a beginner full-stack developer. The goal was to create a clean, responsive, and modern website that represents my personal brand while keeping the structure simple and professional.

The website was built using HTML, CSS, and JavaScript. I focused on layout structure, responsive design, smooth scrolling effects, and organised content sections.

This project helped me practise building a complete website from scratch — from planning the layout to designing sections and refining the user experience.

🎨 Design & Components

I created a reusable card system to display both skills and projects. This made the design consistent and easy to maintain. Each card includes a title, description, and interactive hover effects to improve user engagement.

I also implemented:

  • Scroll fade-in animations
  • Sticky side navigation
  • Active navigation highlighting
  • Responsive grid layouts
  • Clean spacing and typography

These design decisions helped make the website visually organised and modern.

🛠 Technologies Used

  • HTML5
  • CSS3 — Flexbox & Grid
  • JavaScript
  • Bootstrap 5
  • Responsive Design Principles

📚 What I Learned

  • Building this portfolio taught me:
  • How to structure a complete multi-section website
  • How to create reusable UI components
  • How to implement scroll animations and dynamic navigation
  • How to manage layout issues — positioning, overflow
  • The importance of clean code structure and readable styling
  • How small UI details improve user experience

This project represents my growth and confidence as a developer. It also serves as a foundation that I will continue improving as I learn more advanced technologies.
